override ConditionStatusType ConditionSlotFilled.ConditionFulfilled ( List< MercaturaObject > listToCheck)

Проверяет выполнение условия, т.е. находится ли нужный предмет в слоте

Parameters
listToCheckСписок объектов для проверки.
Returns
ConditionStatusType.True, если предмет разрешенного типа надет в указанном слоте; иначе ConditionStatusType.False. Если listToCheck равно null, возвращает ConditionStatusType.Nan.
Exceptions
ArgumentExceptionВыбрасывается, если listToCheck не содержит элемента на позиции CheckPosition или элемент на этой позиции не является CreatureTraits.
