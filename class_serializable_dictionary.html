Detailed Description

Представляет сериализуемый словарь, который может использоваться в Unity. Этот класс позволяет хранить пары ключ-значение, которые можно сериализовать для сохранения состояния.

TKeyТип ключа в словаре.
TValueТип значения в словаре.
