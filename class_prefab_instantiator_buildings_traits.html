Detailed Description

Класс черты, показывающей что предмет может инстанцировать (создавать) префабы зданий на игровой сетке. Имеет соответствующий функционал для создания зданий Наследуется от базового класса PrefabInstantiatorTraits.

Создает префаб здания на указанных координатах и с заданным поворотом.

Parameters
userСущество, которое инициирует создание здания.
spawnCoordinatesКоординаты, где будет размещен префаб здания.
rotationУгол поворота здания при его создании.
Exceptions
ArgumentExceptionБросается в следующих случаях:
  1. Если объект пересекается с другими объектами на игровой сетке.
  2. Если координаты спауна находятся за пределами земель, принадлежащих пользователю.
