Detailed Description

Класс PortableSlotUIData представляет собой структуру данных, используемую для хранения информации о слоте предметов в интерфейсе. Он включает в себя размер слота и данные иконки предмета, а также метод для создания и загрузки экземпляра PortableSlotUI.
