Detailed Description

Utility class to allow the ModelAsset file for an agent to be overriden during inference. This is used internally to validate the file after training is done. The behavior name to override and file path are specified on the commandline, e.g. player.exe –mlagents-override-model-directory /path/to/models.

Additionally, a number of episodes to run can be specified; after this, the application will quit. Note this will only work with example scenes that have 1:1 Agent:Behaviors. More complicated scenes like WallJump probably won't override correctly.
