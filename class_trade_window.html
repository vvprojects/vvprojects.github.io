Detailed Description

Класс TradeWindow представляет окно торговли, позволяющее пользователям просматривать и управлять ордерами на покупку и продажу товаров. Этот класс обновляет интерфейс пользователя в ответ на изменения в ордерах, а также предоставляет методы для создания ордеров.
